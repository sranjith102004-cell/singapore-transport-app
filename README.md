Singapore Transport App
A mobile app project for Singapore commuters, built to make bus and MRT/LRT information easier to access in one place.

Overview
I started this app because I felt many transport apps provide basic information but can be difficult to navigate or may not include all the features commuters use often. My goal is to create a cleaner and more user-friendly app that combines bus arrivals, MRT/LRT information, favourites, trip planning, route tracking, and map-related features.

The app is still under development and is not fully completed yet. However, many core features have already been implemented and tested.
