# MyBusApp

An Expo app for nearby Singapore bus arrivals, saved stops, and MRT/LRT reference data.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Configure the LTA DataMall key

   Create `.env.local` in the project root:

   ```bash
   EXPO_PUBLIC_LTA_API_KEY=your_lta_datamall_key
   ```

3. Start the app

   ```bash
   npx expo start
   ```

You can open the app in Expo Go, an Android emulator, an iOS simulator, or on web with:

```bash
npm run web
```
